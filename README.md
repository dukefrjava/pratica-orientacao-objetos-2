# Pratica orientação à objetos 2


1. Crie o objeto Aluno, com atributos e métodos de sua preferência.

2. Crie um vetor "nota" de n posições, representando a quantidade de provas que os alunos fizeram no ano.

3.  Permita que um usuário crie n Alunos e digite os valores das notas que cada aluno tirou durante o ano.

4.  Ao final, exiba os dados de cada aluno e as médias de cada um. As médias representam a soma das notas dividido pela quantidade de notas.

5.  Crie cada aluno com parâmetros de construtores de sua escolha. O objeto aluno deve seguir os padrões de modificadores de acesso vistos no módulo para a construção de um POJO.

6.  Não permita que um aluno tenha uma nota menor do que zero ou maior do que 10. Informe o usuário que ele digitou uma nota inválida e precisa repreencher a nota em caso de invalidez.
